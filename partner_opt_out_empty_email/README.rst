==============================
Partner Opt-Out if empty email
==============================

This module add a new condition to "Not Opt-Out" filter for partners in order
to filter out also partners with empty email.

This filter is commonly used when selecting partners for a mass mailing.
Now partners with Opt-Out = True or no email defined are excluded
